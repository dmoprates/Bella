# Bella
Página criada para treinamento em desenvolvimento Web utilizando HTML e CSS.
Essa página foi desenvolvida tomando por base o modelo da página AnnaBella 
apresentado no curso da Udemy - Domine Web - 20 Cursos - HTML5, CSS3, SASS, 
Bootstrap, JS, ES6, PHP, OO, MySQL, JQuery, MVC, APIs, IONIC e muito mais

## Competências desenvolvidas
 * Marcação HTML de uma página simples;
 * Criação de páginas para navegação;
 * Estilização do conteúdo
